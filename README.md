# Landing Page - Agência XPTO

Este projeto é uma landing page simples e elegante para a Agência XPTO, desenvolvida com HTML, CSS e JavaScript. A página apresenta uma estrutura moderna com navegação, um painel interativo de projetos e uma seção de apresentação da agência.

## Objetivo

Criar uma interface visual atrativa para apresentar a marca, seus projetos e informações de contato de forma clara e responsiva.

## Funcionalidades

- Header com logo e navegação para seções da página
- Painel de projetos com navegação por botões de avançar e voltar
- Seção “Quem somos” com conteúdo e imagem ilustrativa
- Rodapé com logo, ícones de redes sociais e copyright
- Layout responsivo para diferentes tamanhos de tela

## Estrutura do projeto

- [index.html](index.html): estrutura principal da landing page
- [src/css/estilo.css](src/css/estilo.css): estilos e responsividade
- [src/js/painel.js](src/js/painel.js): lógica do painel de imagens
- [src/imagens](src/imagens): imagens utilizadas no projeto

## Como visualizar o projeto

Você pode abrir o arquivo [index.html](index.html) diretamente no navegador.

Se preferir, pode servir a pasta localmente com um servidor simples, por exemplo:

```bash
python3 -m http.server 8000
```

Depois, acesse:

```text
http://localhost:8000
```

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)

## Observação

Este projeto foi desenvolvido como uma landing page estática, sem necessidade de instalação de dependências adicionais.
